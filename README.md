# Этот проект представляет собой веб-сайт турагентства, разработанный с использованием Flask. Сайт позволяет пользователям регистрироваться, входить в систему и просматривать доступные туры.

# Установка
Требования
Python 3.x
Flask
Flask-SQLAlchemy
Flask-Login
Flask-WTF
# Установка зависимостей
Клонируйте репозиторий:

git clone https://github.com/ваш_логин/ваш_репозиторий.git
cd ваш_репозиторий
Установите необходимые пакеты:

pip install -r requirements.txt
# Конфигурация
Создайте базу данных. В файле config.py установите параметры подключения к базе данных.
Инициализируйте базу данных:

python
>>> from app import db
>>> db.create_all()
#Запуск приложения
Запустите сервер:

python run.py
Откройте браузер и перейдите по адресу:

http://127.0.0.1:5000
# Функциональность
Регистрация пользователей: Пользователи могут создавать свои учетные записи.
Логин: Уже зарегистрированные пользователи могут входить в систему.
Просмотр туров: После входа пользователи могут просматривать доступные туры.
Используемые технологии
Python
Flask
SQLite (или другая СУБД по желанию)
HTML/CSS для фронтенда
